SubTrack — Proper MVC Structure
Backend (Java Spring Boot)  +  Frontend (HTML/CSS/JS)  +  REST API Endpoints
Project Root
subtrack/
Backend (MVC Core — Java Spring Boot)
config/settings
DatabaseConfig.java— SQLite datasource & JPA setup
SecurityConfig.java— Spring Security, JWT filter
GroqConfig.java— AI API key & endpoint constants
AppConstants.java— billing cycles, alert thresholds
model/M — Model
User.java— id, name, email, passwordHash, monthlyBudget
Subscription.java— name, category, price, cycle, renewalDate, status
PaymentRecord.java— subscriptionId, amount, date, status
Alert.java— type, message, triggerDate, isRead
Category.java— name (streaming, software, gaming, fitness…)
SpendingSummary.java— month, totalSpent, categoryBreakdown[]
controller/C — Controller
AuthController.java— register, login, logout, profile
SubscriptionController.java— CRUD, list, filter, export
PaymentController.java— history, upcoming, mark-paid
AlertController.java— list alerts, mark-read, dismiss
AnalyticsController.java— monthly spend, category breakdown, trends
AIAdvisorController.java— expense analysis, waste detection, suggestions
BudgetController.java— set budget, get progress, overspend check
routes/REST API endpoints
api/— /api/v1/...
auth.routes— POST /register /login /logout
subscription.routes— GET /subscriptions POST PUT DELETE /:id
payment.routes— GET /payments POST /record GET /upcoming
alert.routes— GET /alerts PATCH /read DELETE /dismiss
analytics.routes— GET /spend/monthly /spend/category /trends
ai.routes— POST /analyze /detect-waste /suggest
budget.routes— GET /budget PUT /budget GET /budget/status
export.routes— GET /export/csv /export/json
middleware/guards
JwtAuthFilter.java— verify JWT on every request
GlobalExceptionHandler.java— @ControllerAdvice error handler
RequestValidator.java— input validation for subscription fields
RateLimitFilter.java— limits AI API calls per user
service/business logic
GroqService.java— calls Groq LLM API, parses JSON response
InsightService.java— builds AI prompts with subscription context + tips
WasteDetectorService.java— detects overlapping & unused subscriptions
SpendingAnalyzerService.java— monthly totals, trend calculation, category grouping
AlertSchedulerService.java— @Scheduled job: fires renewal alerts 3/7 days ahead
FileExportService.java— exports subscription list to .csv / .json
UserService.java— auth logic, password hashing, JWT generation
util/helpers
ApiResponse.java— standardized JSON response wrapper
DateUtil.java— renewal countdown, billing cycle helpers
CurrencyUtil.java— PKR / USD formatting
JwtUtil.java— JWT sign & verify
Frontend (HTML/CSS/JS — V in MVC)
static/V — View
pages/
login.html— login / register tabs
dashboard.html— summary cards: total spend, renewals due, waste found
subscriptions.html— table of all subscriptions + add/edit/delete
analytics.html— monthly bar chart + category pie chart (Chart.js)
ai-advisor.html— chat-style AI expense advisor panel
alerts.html— renewal reminders + budget warnings list
payments.html— payment history timeline per subscription
budget.html— budget goal setter + monthly progress bar
profile.html— user settings, currency preference, export
partials/
navbar.html
subscription-card.html
alert-badge.html
Static Assets
public/static files
css/
main.css
dashboard.css
js/
subscriptions.js— CRUD calls, table render
analytics.js— Chart.js bar + pie charts
ai-advisor.js— chat UI, fetch /api/analyze
alerts.js— poll for new alerts, show banners
auth.js— login/register, save JWT to localStorage
images/
logo.png
service-icons/— Netflix, Spotify, Adobe logos etc.
Database & Data Files
data/SQLite + files
subtrack.db— auto-created SQLite database
exports/— user CSV/JSON subscription exports
Resources (AI Knowledge Base)
resources/AI context files
subscription_tips.txt— category-wise saving tips for AI prompts
category_rules.txt— rules for waste/overlap detection logic
Root Files
SubTrackApplication.java— Spring Boot main entry point
application.properties— GROQ_API_KEY, SQLite path, JWT secret
.env.example
pom.xml— Spring Boot, SQLite, org.json, Spring Security
