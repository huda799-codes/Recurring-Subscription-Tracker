How the web version works — subscription tracker data flow
User (Browser) → adds subscription or asks AI advisor
    ↓  fetch POST /api/subscriptions or /api/ai/analyze
Java Spring Boot Backend → receives request
    ↓  SubscriptionController saves to SQLite via JPA
    ↓  InsightService loads context from subscription_tips.txt
    ↓  GroqService calls Groq API with subscription data + context
    ↓  AlertSchedulerService checks renewal dates in background
    ↓  FileExportService writes subscription CSV to data/exports/
    ↓  returns JSON { "insight": "...", "wastFound": [...] }
Browser → displays AI insight + updates charts ✅
